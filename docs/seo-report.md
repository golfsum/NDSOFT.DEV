# ND SOFT homepage SEO report

Research date: July 20, 2026

## Intent and positioning

This page serves branded and navigational intent for ND SOFT, AppsResolve, Tranqly, PawProof, and TeeLesson. It is intentionally a studio index rather than a broad software-services page. Application-support search intent belongs to AppsResolve and is routed there directly.

## Source checks

- `https://appsresolve.com/` confirms AppsResolve as human-reviewed application support for SaaS teams.
- `https://pawproof.app/` confirms pet record scanning, care records, and reminders.
- The local Tranqly product contract confirms one-minute daily reflection and pattern discovery.
- The local TeeLesson product contract confirms student, lesson, practice, progress, and coaching operations.

## Technical implementation

- Canonical origin: `https://ndsoft.dev/`
- One descriptive title and one H1
- Server-rendered product copy and descriptive outbound links
- Organization and ItemList structured data matching visible content
- Open Graph and X metadata with a first-party social card
- Sitemap, robots policy, web manifest, theme color, and custom not-found page
- No thin secondary routes or duplicated product acquisition content

## Remaining production checks

The final public origin, HTTP redirects, live response headers, Search Console submission, and production Core Web Vitals require deployment access and are not verified locally.
